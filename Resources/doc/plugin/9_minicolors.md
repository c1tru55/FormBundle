### MiniColors

Homepage: http://labs.abeautifulsite.net/jquery-minicolors/

Provided field types:

| Type                | Parent type | Required components |
|---------------------|-------------|---------------------|
| ite_minicolors_text | text        | none                |

Example configuration:

```yml
# app/config/config.yml

ite_form:
    plugins:
        minicolors: ~
```
